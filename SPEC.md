# AI 内容重制工具 - MVP 规范

## 核心价值主张
把长文章/视频转成社交媒体帖子、邮件、短信 - 营销刚需

## 验证问题
- [x] 可以一句话描述问题：内容创作者需要把长内容转成短格式
- [x] 目标客户：营销人员、内容创作者、创始人

## 技术栈
- Next.js 15 + TypeScript
- Tailwind CSS
- 内置 prompt 模板（无需外部 AI API）

## MVP 功能
1. 输入：粘贴文本
2. 选择输出格式：Twitter线程、LinkedIn帖子、冷邮件、Instagram caption
3. 基于模板的重制
4. 一键复制
